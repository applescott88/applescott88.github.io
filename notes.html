GitHub Pages 架站完整流程
2025-01
使用 GitHub Pages 可以免費架設靜態網站。流程：建立 repository → 命名為 username.github.io → 上傳 index.html → 在 Settings/Pages 啟用部署。網站約 1-2 分鐘後上線，支援自訂網域。

訓練區間心率筆記（Zone 2 訓練）
2025-01
Zone 2 定義為最大心率的 60-70%。長時間 Zone 2 騎乘能提升有氧基礎、增加粒線體密度。建議每週 3-4 次、每次 60 分鐘以上。談話測試：能說完整句子但略感喘息即為 Zone 2。
